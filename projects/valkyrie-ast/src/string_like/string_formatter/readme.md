`"string formatter \n\r {expr:args} \u{1234} "`

Aka. String Interpolation

| Escape Sequence | Meaning               |
|-----------------|-----------------------|
| `\n`            | Line Feed             |
| `\r`            | Carriage Return       |
| `\t`            | Horizontal Tab        |
| `\v`            | Vertical Tab          |
| `\0`            | Null                  |
| `\\`            | Backslash             |
| `\"`            | Double Quote          |
| `\'`            | Single Quote          |
| `\u{1234}`      | Unicode Character     |
| `\x12`          | Hexadecimal Character |
| `\o123`         | Octal Character       |
| `\b`            | Backspace             |
